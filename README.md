# Music-Recommendation-Website
A web-based application that serves personalized recommendations to users. Users can select the recommendation model type (e.g., popularity-based, cosine similarity, k-means, neural networks) and input their preferences. The system then generates tailored suggestions with corresponding links, providing an interactive experience.
